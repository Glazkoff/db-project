## Технологии

- Docker
- PostgreSQL
- Flask
- Nginx
- Gunicorn

## Разработка

### Перед началом работы

- В корне проекта необходимо создать файл `.env` по формату `.env.example` и заполнить переменные окружения

### Необходимые команды

`docker compose up --build` - запуск проекта
